# Threat_Detection_System

OpenCV와 딥러닝 기반의 실시간 위협 탐지 시스템입니다.

실시간 카메라 영상을 분석하여 사람 객체를 탐지하고 추적하며, 잠재적인 위협 대상을 시각화하는 컴퓨터 비전 프로젝트입니다.

향후 모바일 카메라 등을 통해 영상을 촬영하고 PC로 영상을 처리하여 모바일 화면에 출력해서 고글형 AR과 유사한 기능을 추가하는 것이 최종 목표입니다

## 주요 기능
- 실시간 카메라 영상 처리
- 사람 객체 탐지 및 추적
- 위협 대상 분석 및 시각화
- HUD 스타일 정보 오버레이
- OpenCV 기반 영상 처리 파이프라인
